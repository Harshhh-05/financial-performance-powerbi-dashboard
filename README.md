# Financial Performance & Ratio Analysis Dashboard

##  Project Overview
This project analyzes the financial performance of major Indian companies using **Power BI**, focusing on profitability, liquidity, leverage, efficiency, and growth trends.  
The dashboard is designed to support **executive-level decision-making** and **comparative financial analysis**.

---

##  Companies Analyzed
- Reliance Industries  
- TCS  
- Infosys  
- HDFC Bank  
- ITC
